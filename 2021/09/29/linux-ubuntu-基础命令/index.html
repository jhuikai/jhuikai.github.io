linux ubuntu 基础命令

2021-09-29

Ubuntu是一个以桌面应用为主的 Linux 操作系统，以下信息主要关于使用ubuntu的基础命令。

仅供参考！

一、文件及文件夹管理

ls 查看当前目录文件（不包含隐含文件）
ls -a 列出当前目录文件（包含隐含文件）
ls -l 查看当前文件以列表形式显示（不包含隐含文件）
ls -al 以列表形式查看当前所有的隐含文件
ls -hl 更加人性化显示文件
cd 文件名 进入文件夹
cd ~ 回到自己的home文件夹
cd . 当前目录
cd … 回当前目录的上一级目录
cd - 回上一次所在的目录

pwd 查看自己当前所在文件夹路径

mkdir 创建文件夹

mkdir 文件夹名

mkdir 文件夹名/文件夹名/…在一个文件夹下创建文件夹

mkdir 文件夹名/文件夹名/…-p 新文件夹加-p可以直接创建所有文件夹

touch 创建文件

touch 文件名.文件后缀

rm 删除命令

rm -i 文件名.文件后缀 询问是否删除 yes / no

![](linux-ubuntu-基础命令/rm -i.png)

rm -f 文件名.文件后缀 不询问直接删除

![](linux-ubuntu-基础命令/rm -f.png)

rm -r 文件名.文件后缀 递归删除文件夹 删除所有文件和文件夹 不询问

![](linux-ubuntu-基础命令/rm -r.png)

rm -ri 文件名.文件后缀 递归删除文件夹 删除所有文件 询问是否删除 yes / no

![](linux-ubuntu-基础命令/rm -ri.png)

mv 移动文件

mv 路径/文件 /经/文件移动相对路径下的文件到绝对路径下

mv 文件名 新名称 在当前目录下改名

find 查找文件

find -name ‘文件名’ 搜索所有

find ./文件夹 -size -2M 搜索所有小于2M的文件 +2表示大于

cp 复制文件

cp 文件夹/文件名 ./移动的地址

tar 打包文件

tar -cvf 打包文件名.tar 要打包的文件或文件夹1.txt 2.txt

ar xvf打包文件名.tar 解开打包文件

tar -cvf 打包文件名.tar.gz 压缩要打包的文件或文件夹1.txt

ar -zxvf 打包文件名.tar.gz 解开压缩包文件

二、系统管理

reboot lnit 6 重启LINUX系统

Halt lnit 0 Shutdown -h now 关闭LINUX系统

三、apt命令

apt-cache search package 搜索包
apt-cache show package 获取包的相关信息，如说明、大小、版本等
sudo apt-get install package 安装包
sudo apt-get install package - - reinstall 重新安装包
sudo apt-get -f install 修复安装"-f = -fix-missing"
sudo apt-get remove package 删除包
sudo apt-get remove package - - purge 删除包，包括删除配置文件等
sudo apt-get update 更新源
sudo apt-get upgrade 更新已安装的包
sudo apt-get dist-upgrade 升级系统
sudo apt-get dselect-upgrade 使用dselect升级
apt-cache depends package 了解使用依赖
apt-cache rdepends package 是查看改包被哪些包依赖
sudo apt-get build-dep package 安装相关的编译环境
apt-get source package 下载改包的源代码
sudo apt-get clean && sudo apt-get autoclean 清理无用的包
sudo apt-get check 检查是否有损坏的依赖
sudo apt-get clean 清理所有软件缓存（即缓存在/var/cache/apt/archives目录里面的deb包）

四、vi编辑器的一些操作指令

运行C语言文件：gcc 文件名.c && ./a.out

五、more分屏显示

当出现信息过多快速滚屏时，用户会出现无法看清文件的问题，此时可以使用more命令，每次只显示一页，按下空格键可以显示下一页，按下q键退出显示，按下h键可以获取帮助。

命令：more 文件名.文件后缀

ctrl + b 返回上一屏

ctrl + f 下一屏
