字符串的格式化

PHP提供了chop()、ltrim()和trim()三个函数，用来删除各种各样的多余空格或者指定的内容。

chop()

chop()还有一个同义函数rtrim()，两者用法完全相同、作用完全一样。

  • 语法：chop(string,charlist)

string：必需，规定要转换的字符串； charlist：可选，规定从字符串删除哪些字符。

  • 作用：去除字符串尾部的空格等特殊字符，如果有使用charlist指定特定字符，则去除指定的字符。

  • 语法：ltrim(string,charlist)

  • 用法：去除字符串头部的空格等特殊字符，如果有使用charlist指定特定字符，则去除指定的字符。

  • 语法：trim(string,charlist)

  • 用法：去除字符串头部和尾部的空格等特殊字符，如果有使用charlist指定特定字符，则去除指定的字符。

substr()

  • 语法：substr(string,start,length)

    参数 描述
    start 必需，要抽取的字符串的起始下标。必须是数值。如果是负数，那么该参数声明从字符串的尾部开始算起的位置。也就是说，-1 指字符串中最后一个字符，-2 指倒数第二个字符，以此类推。
    length 可选，子串中的字符数。必须是数值。如果省略了该参数，那么返回从 stringObject 的开始位置到结尾的字串。

  • 用法：在字符串中抽取从 start 下标开始的指定数目的字符。

  • 例子：

  • <?php
echo substr('abcdef', 1); // bcdef
echo substr('abcdef', 1, 3); // bcd
echo substr('abcdef', 0, 4); // abcd
echo substr('abcdef', -1, 1); // f
echo substr('abcdef', -3, 1); // d
echo substr('abcdef', 4, -4); // 空值
echo substr('abcdef', 2, -1); // cde
echo substr('abcdef', -3, -1); // de
?>
